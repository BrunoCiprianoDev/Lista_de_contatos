Autor: Bruno Cipriano
Repositório: https://githun.com/BrunoCiprianoDev/Lista_de_contatos

Descrição:
O projeto Lista_de_contatos é um aplicação focada na implementação de um CRUD para perpetuação de informações em um bancos de dados SQL. Com base nessa premissa, por meio de uma interface simples e intuitiva o aplicativo permite que o usuário registre seus contatos, compostos por nome e telefone, em um lista ordenada.
